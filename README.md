# pinyin-converter
Converts Mandarin Chinese pinyin without tone marks to pinyin with tone marks.

Input:
Pinyin with number suffixes separated by spaces, ex. `zao3 shang4 hao3 zhong1 guo2`

Output:
Pinyin with tone marks, ex. `zǎo shàng hǎo zhōng guó`

Run in terminal with `python3 [path to pinyin.py]`
